🔍 Comprobar dashboard (Silber)

Opción rápida (hazlo tú mismo):

  1. Abre la app (index.html), inicia sesión.
  2. Pulsa F12 → pestaña Consola.
  3. Pega este código y pulsa Enter:
var d=document.getElementById("screen-dashboard");if(d){d.style.display="block";d.style.visibility="visible";d.style.opacity="1";var c=d.querySelector(".dashboard-content");if(c){c.style.display="flex";c.style.visibility="visible";}console.log("OK: dashboard forzado");}else{console.log("No hay #screen-dashboard");}

Si después de pegarlo ves contenido en el centro, el problema es de CSS y ya está corregido en esa sesión.

Alternativa: Abre la app en una pestaña, luego este archivo en otra y pulsa el botón.
